## Requirements


- [Node v18.12.1](https://nodejs.org/en/)
- [Docker version 20.10.20](https://www.docker.com/products/docker-desktop/)

## Clone Repository

```bash
$ git clone git@github.com:jhonsstn/nest-api.git
$ cd nest-api
```

## Install locally

```bash
$ npm install
```

## Running the app on docker

```bash
# starting
$ npm run compose:up

# stopping
$ npm run compose:down
```

## Install locally

```bash
$ npm install
```

## Running the app locally

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Documentation

```bash
# after project is running (if running on default port).
http://localhost:3000/api
```
